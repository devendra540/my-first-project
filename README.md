# My First DevOps CI/CD Project

## Project Description

This project is created to learn a complete DevOps CI/CD workflow using Azure DevOps and GitHub.

The project demonstrates how source code is stored, built, tested, containerized, and deployed automatically.

---

## Technologies Used

- Java
- Maven
- Git
- GitHub
- Azure DevOps
- Docker
- Kubernetes
- SonarQube
- YAML Pipeline
